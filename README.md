# 2kW Load Driver - High-Power LED Control Circuit

A professional-grade 2000W LED driver circuit designed for industrial applications. This project demonstrates advanced power electronics design including half-bridge MOSFET topology, thermal management, and microcontroller-based PWM control.

## Project Overview

**Specifications:**
- Input: 40V DC
- Output: 36V DC, 56A continuous (2000W)
- Peak Current: 100A
- Control: PWM dimming via microcontroller
- Switching Frequency: 20kHz
- Temperature Monitoring: 4x NTC sensors

**Key Features:**
- ✅ Half-bridge MOSFET topology for efficient power delivery
- ✅ Microcontroller-based PWM control with dimming
- ✅ Thermal protection with NTC temperature sensors
- ✅ Control isolation via optocoupler
- ✅ Professional PCB layout (120×90mm, 2-layer, 3oz copper)
- ✅ Output LC filter for ripple reduction
- ✅ Bootstrap circuit for high-side gate drive
